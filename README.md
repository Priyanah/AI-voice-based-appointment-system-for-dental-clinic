# AI-voice-based-appointment-system-for-dental-clinic

I have developed an AI voice based receptionist using N8N, docker, retell (for voice).

This AI receptionist takes takes the call book a full appointment uaing patient detials.

It also checks whether the patient has recently taken an treatment from the clinic.

It also give follow ups on the latest treatments, also handles after treatment problems like if any pain occurred to the patient after treatment it also handles them, like asking for follow like is patient taking the prescribed medicine by the doctor. and also handels basic problems by itself not involving doctors.

It checks existing patient using mobile number.

It also books/update the appointment in Calendar and sheets.
